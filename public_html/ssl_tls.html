SSL & TLS Protocols



Secure Socket Layer (SSL) and Transport Layer Security (TLS), both are cryptographic protocols that provide authentication to a user while logging in and data encryption between servers, machines, and applications operating over a network (client connecting to a web server). It is the standard technology for keeping an internet connection secure and safeguarding any sensitive data that is being sent between two systems, preventing criminals from reading and modifying any information transferred, including potential personal details

The two systems can be a server and a client (shopping website and browser) or server to server (application with personal identifiable information or with payroll information).

They are security layers that protect the website which are further maintained by installing and configuring Digital Certificates.
